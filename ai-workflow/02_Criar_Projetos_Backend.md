# 02 — Criar Projetos Backend

## Objetivo
Definir e criar os projetos que compõem o backend (APIs, core, infra, testes).

## Passos recomendados
- Enumerar projetos: `FilmFlix.Api`, `FilmFlix.Core`, `FilmFlix.Infrastructure`, `FilmFlix.Tests`.
- Criar os projetos vazios (esqueleto) sob a solution existente.
- Não implementar lógica de negócio — apenas estrutura de pastas e projetos.

## Critérios de aceitação
- Cada projeto aparece na solution.
- Estrutura de pastas e README básico criado.
