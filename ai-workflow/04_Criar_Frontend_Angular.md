# 04 — Criar Frontend Angular

## Objetivo
Preparar a estrutura inicial do frontend em Angular.

## Passos recomendados
- Criar o workspace Angular minimal: `nx` ou `ng` scaffold, mas apenas estrutura (não componentes completos).
- Definir arquitetura de módulos e contrato com backend (endpoints, DTOs).
- Incluir README com comandos de build e run.

## Aviso
Não implementar telas completas; parar para revisão de arquitetura antes de avançar.
